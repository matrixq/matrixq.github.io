实在受不了WordPress的繁琐和上传图片的方式，于是转到了Markdown；也没有什么流量，时写时不写，域名和空间的管理也麻烦，所以转到Github，希望即使长草的时间很长也可以很容易的找回以前的文章，至少可以当作笔记。
